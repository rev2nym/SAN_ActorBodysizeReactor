# SAN_ActorBodysizeReactor
このスクリプトはRPGツクールMV向けプラグインです。

## 概要
このプラグインには次の2つの機能を実装しています。

- ボディサイズ反映機能  
アクターに性別、年齢、身長、体重、スリーサイズ等のパラメータを追加し、
それらボディサイズに職業、装備、スキル、ステートを反映する機能です。

- データ編集機能  
ボディサイズ反映機能のデータを編集する機能です。
